OutputRenderer 输出渲染器

输出模式渲染器。调用 OutputRenderer 的 render() 方法，可以让 Textbus 以输出模式渲染文档。通过输出模式渲染文档，不会触发组件的生命周期，也不会渲染真实页面的内容，只会在内存中生成虚拟 DOM 树。

通过编辑器实例或通过 useContext 勾子获取 OutputRenderer 的实例。

// 方法1
const outputRenderer = editor.get(OutputRenderer)
// 方法2
const outputRenderer = injector.get(OutputRenderer)
// 方法3
const yourComponent = defineComponent({
  setup() {
    const outputRenderer = useContext(OutputRenderer)
  }
})
TypeScript

render() 渲染

以输出模式渲染文档内容，并返回渲染后的虚拟 DOM 树。

// 类型签名：
render(): VElement;
TypeScript
