开始

Textbus 是一个以组件为中心，数据驱动的富文本编辑器！为了解决传统富文本开发中遇到的各种问题，我们完全自主设计了富文本的组件系统和格式系统，旨在降低富文本的开发成本，帮助你更轻松的扩展出丰富的富文本内容编辑功能。不管你是富文本开发的老鸟，亦或是前端的新手，你都将会在 Textbus 中感受到非凡的开发体验和扩展能力。

你也许已经使用过 Vue 或者 React 这样的前端框架，你是否惊叹于这些前端框架带给你革命性的开发体验？

如果答案为是的话，那么你将在 Textbus 里获得同样的感受。是的，当你看到 Textbus 的组件扩展时，你会发现，这和前端框架太像了。你可以通过 defineComponent 函数定义一个组件，再用 jsx 写 render，简直再熟悉不过了。这在以前是不敢想象的。现在，你唾手可得。

让我们开始吧！

安装

通过 npm 安装 Textbus：

npm install @textbus/editor @textbus/core @textbus/browser

准备好一个 DOM 元素。

<div id="editor"></div>
HTML

创建一个编辑器。

import "@textbus/editor/bundles/textbus.min.css"
import { createEditor } from '@textbus/editor'

const editor = createEditor(document.getElementById('editor'))

// 监听数据变化，并获取编辑结果
editor.onChange.subscribe(() => {
  const contents = editor.getContents()
  console.log(contents)
})
TypeScript

现在我们的编辑器就创建好了。

通过标签引入

如果你是传统的项目，你可以通过标签引入。

点击下载 textbus.m​in.js textbus.m​in.css。

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="https://textbus.io/lib/textbus.min.css">
  <script src="https://textbus.io/lib/textbus.min.js"></script>
</head>
<body>
  <div id="editor"></div>
  <script>
    var editor = textbus.editor.createEditor(document.getElementById('editor'))
    editor.onChange.subscribe(function() {
      const contents = editor.getContents()
      console.log(contents)
    })
  </script>
</body>
</html>
Jsx
